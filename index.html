Forward Diffusion: Adding Gaussian Noise

xnoisy = (1 - t) · x + t · z

Where:

  • x = original image (each pixel is an RGB vector, values in [0, 1])
  • z ~ N(0, I) = Gaussian noise (each pixel gets 3 independent random values from a standard normal distribution)
  • t = time step (0 = original image, 1 = pure noise)

Intuition: As t increases from 0 to 1, we linearly interpolate from the original image to pure random noise. At t=1, all information about the original image is destroyed — it becomes indistinguishable from random static.

Why this matters: Diffusion models learn to reverse this process. They're trained to predict and remove the noise, so they can generate images by starting from pure noise (t=1) and gradually denoising back to t=0.
